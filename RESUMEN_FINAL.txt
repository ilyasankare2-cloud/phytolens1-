PASOS PARA SUBIR A GITHUB
════════════════════════════════════════════════════════════════════════════════

1. Crear cuenta en GitHub (si no tienes): https://github.com/signup

2. Crear nuevo repositorio:
   - Nombre: phytolens
   - Descripción: "Sistema de IA para clasificación de plantas"

3. Desde PowerShell (en el directorio del backend):

   git init
   git add .
   git commit -m "PhytoLens: Sistema IA completado"
   git remote add origin https://github.com/TU_USUARIO/phytolens.git
   git branch -M main
   git push -u origin main

4. Listo! Tu proyecto está en GitHub
